# 🛒 SmartCart – College Shopping Management System

SmartCart is a Streamlit-based web application that simulates a basic shopping management system with separate **Admin** and **Customer** roles.

This project demonstrates session management, inventory handling, cart system, revenue tracking, and interactive UI design using Streamlit.

---

## 🚀 Features

### 👨‍💼 Admin Panel
- Secure login (Username: `admin` | Password: `admin`)
- View total revenue
- View total products
- Monitor product inventory
- Logout functionality

### 🛍 Customer Panel
- Browse available products
- Add products to cart
- Live stock update
- View cart items
- Apply 10% discount at checkout
- Confirm order
- Automatic revenue update

---

## 🖥️ Technologies Used

- Python
- Streamlit
- PIL (Python Imaging Library)
- Session State Management
- Custom CSS Styling
